# Objetivos:
O principal objetivo deste projeto é desenvolver, modelar e implementar o software RuyRotas,
que é um "Sistema de Gerenciamento e Monitoramento de Rotas de Transporte Escolar Municipal" 
voltado para a cidade de Ruy Barbosa. 
Queremos implementar e criar um sistema voltado para a logística estudantil que permita com que seus utilizadores, 
como alunos da rede pública e seus responsáveis, tenham acesso a diversos recursos que os ajudem a planejar seus deslocamentos diários com maior previsibilidade e segurança. 
O sistema visa preencher a falta de dados centralizados por meio do mapeamento de linhas, definição de paradas e acompanhamento dos ônibus em tempo real.
